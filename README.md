# ESP32 Smart Home

A Smart Home Automation System built using ESP32 and ESP-IDF.

## Features

- WiFi Access Point
- Web Dashboard
- Control 2 Relays
- Live Status
- Auto Refresh
- Responsive UI

## Hardware

- ESP32 DevKit V1
- 2 Channel Relay Module

## Software

- ESP-IDF
- C
- HTML
- CSS

## Future Improvements

- HTTPS Server
- Voice Control
- Mobile App
- Sensor Monitoring
